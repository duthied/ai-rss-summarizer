# AI RSS Summarizer

Cost-optimized RSS feed digest generator using Claude AI.

## Setup

1. Clone and install:
   ```bash
   git clone https://github.com/duthied/ai-rss-summarizer.git
   cd ai-rss-summarizer
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

2. Configure:
   ```bash
   cp .env.example .env
   # Edit .env and add your ANTHROPIC_API_KEY
   ```

3. Run:
   ```bash
   python rss-implementation-quick.py
   ```

## Cost

~$0.05 per run, ~$1.50/month for daily digests.
